# Project-Wide Instructions

## Context
This is an AI Sales CRM for the Notion MCP Challenge. Read `.github/copilot-instructions.md` for full project context before making changes.

## Critical Rules
1. NEVER use ESLint — this project uses **Biome** for linting and formatting
2. NEVER use `@google/generative-ai` — use `@google/genai` (the new SDK)
3. NEVER use `database_id` in MCP tool calls — use `data_source_id` (v2 breaking change)
4. NEVER use Edge runtime — always `export const runtime = 'nodejs'`
5. NEVER expose API keys or tokens to the client — all env vars are server-side only
6. Use `@/` import alias for all internal imports
7. Use shadcn/ui components from `@/components/ui/` — don't recreate them
8. Use Tailwind utilities only — no inline styles, no CSS modules
