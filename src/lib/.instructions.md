# Library Code Instructions

## MCP Client (`mcp-client.ts`)
- Module-level singleton pattern — one client instance shared across all requests
- Use `StreamableHTTPClientTransport` (not `StdioClientTransport`)
- Connect to `process.env.MCP_SERVER_URL` (defaults to `http://localhost:3001/mcp`)
- Handle reconnection: if client is disconnected, reconnect transparently
- Export `getMcpClient()` async function that returns the connected client

## Gemini Helper (`gemini.ts`)
- Use `@google/genai` SDK with `GoogleGenAI` class
- Model: `gemini-2.5-flash`
- Export `generateWithTools(prompt: string)` that wires `mcpToTool(mcpClient)` automatically
- Export `generateText(prompt: string, systemPrompt?: string)` for simple generation without tools
- Never stream responses in API routes — return complete results

## Notion Schema (`notion-schema.ts`)
- Single source of truth for all Notion database IDs and property names
- Export typed constants for each database's property names
- All DB IDs come from environment variables with validation
